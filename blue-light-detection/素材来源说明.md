# 素材来源说明

本项目已将原来的自制合成演示素材替换为真实道路/车辆素材，便于验证 YOLO 车辆检测、蓝光分割和视频逐帧处理流程。

## 图片素材

文件：

- `input/night_vehicle_road_0000.jpg`
- `input/night_vehicle_road_0001.jpg`
- `input/night_vehicle_road_0002.jpg`

来源：ntnu-arl/vehicles-nighttime 公开仓库样例图片。

用途：夜间道路车辆检测验证，适合测试低照度场景下的车辆框检测能力。

文件：

- `input/daytime_traffic_bus.jpg`

来源：Ultralytics YOLOv5 公开样例图片。

用途：白天车辆/公交场景验证，适合确认 YOLO 模型和图片处理流程是否正常。

## 视频素材

文件：

- `input/traffic_vehicle_detection.mp4`

来源：jjw-DL/YOLOV3-SORT 公开仓库交通检测样例视频。

用途：真实交通视频流处理验证。该视频可检出多辆车辆，适合测试 `main.py` 的视频逐帧检测与结果导出流程。

## 运行示例

```bash
python main.py --input input/daytime_traffic_bus.jpg --output output/result.jpg
python main.py --input input/night_vehicle_road_0000.jpg --output output/night_result_0000.jpg
python main.py --input input/traffic_vehicle_detection.mp4 --output output/result.mp4
```
