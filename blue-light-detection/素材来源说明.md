# 素材来源说明

本项目已将原来的自制合成演示素材替换为真实道路/车辆图片素材，便于验证 YOLO 车辆检测、车灯候选提取和车辆蓝光关联流程。

## 图片素材

文件：

- `input/night_vehicle_road_0000.jpg`
- `input/night_vehicle_road_0001.jpg`
- `input/night_vehicle_road_0002.jpg`

来源：ntnu-arl/vehicles-nighttime 公开仓库样例图片。

用途：夜间道路车辆检测验证，适合测试低照度场景下的车辆框检测能力。

## 运行示例

```bash
python main.py --input input/night_vehicle_road_0000.jpg --output output/night_result_0000.jpg
```

说明：`main.py` 保留视频逐帧处理函数，但当前提交的作业素材和结果以图片为主，`input/` 目录不再提供视频文件。
