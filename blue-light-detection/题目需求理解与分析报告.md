# 题目需求理解与分析报告

## 高速路高清摄像头蓝光检测

### 一、背景与意义

高速公路卡口、龙门架和隧道监控场景中，高清摄像头会持续采集车辆图像或视频帧。夜间、隧道、雨天和逆光条件下，车灯蓝光、号牌补光、车身反光以及路灯、路牌等环境蓝光会同时出现，人工排查效率低且误判率高。因此，需要结合人工智能目标检测与数字图像处理方法，实现车辆蓝光区域的自动检测、定位和干扰过滤。

### 二、问题界定

- 输入：高速公路监控单帧图像或实时视频流。
- 输出：带有车辆框、蓝光区域框、蓝光来源标签和统计信息的可视化结果。
- 核心目标：识别画面中车辆位置，提取高亮蓝光区域，并判断蓝光是否与车辆相关。
- 干扰过滤：过滤路牌、路灯、天空杂光、路面雨水反光等非车辆蓝光。

### 三、技术挑战

- 夜间和隧道场景中图像噪声高、对比度低，车辆边缘模糊。
- 蓝光既可能来自真实车辆，也可能来自环境反射，单纯颜色阈值容易误检。
- 白天弱蓝光和夜间强蓝光亮度差异大，固定阈值鲁棒性不足。
- 多车重叠、远距离小目标和雨天镜面反光会增加目标关联难度。
- 轻量化部署要求算法在普通 CPU 或低算力设备上也能运行。

### 四、任务拆解

1. 图像预处理：去噪、亮度归一化、局部对比度增强。
2. 蓝光特征提取：HSV 色彩空间阈值分割高亮蓝色像素。
3. 车辆目标检测：使用 YOLOv8n 等轻量模型检测车辆位置。
4. 蓝光关联匹配：计算蓝光区域与车辆检测框的包含比例或 IoU。
5. 蓝光来源判断：根据蓝光区域在车辆框内的相对位置区分左灯、右灯或车身反光。
6. 结果可视化与统计：输出总车辆数、含蓝光车辆数、蓝光区域数和被过滤干扰数。

### 五、评估标准

- 检测正确性：车辆检测框与蓝光区域定位是否准确。
- 误检控制：路牌、路灯、路面反光等干扰是否被过滤。
- 鲁棒性：白天、夜间、隧道、雨天等场景下是否稳定。
- 可解释性：每个判断都有图像处理依据和空间关联依据。
- 运行效率：轻量模型和传统图像处理组合，适合低算力部署。
