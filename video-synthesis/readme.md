# 北航校园视频光流融合

**算法**：智能切点检测 + Farneback 稠密光流 + 双向 Warp + 前向-后向一致性掩码 + 亮度归一化
**过渡帧数**：5 帧（约 0.17s @ 30fps）

## 算法流程：
1. **智能切点检测**：在相邻片段首尾各 60 帧中搜索像素差最小的帧对（8× 降采样加速），作为融合切点
2. **双向 Farneback 光流**：同时计算 A→B 和 B→A 的稠密光流场
3. **前向-后向一致性掩码**：将 B→A 流映射回 A 坐标系，与 A→B 流求和，误差 < 3px 为可信区域；不可信区域退回纯 Alpha 混合，消除鬼影
4. **双向 Warp + 亮度归一化**：可信区域中 f_a 向 B 走 t 步、f_b 向 A 走 (1-t) 步，各走一半降低拉伸；同时对 f_b 亮度做渐进归一化消除颜色突变

## 光流参数：
pyr_scale=0.5, levels=5, winsize=21, iterations=5, poly_n=7, poly_sigma=1.5

| 过渡点 | 最佳切点帧差↓ | 切点帧差↓ | 切点SSIM↑ |
|--------|-------------|---------|----------|
| 1 → 2  | 28.8        | 29.4    | 0.2784   |
| 2 → 3  | 34.2        | 34.6    | 0.3303   |
| 平均   | 31.5        | 32.0    | 0.3043   |

真实数据：3 段北航校园视频（205帧/219帧/181帧，~30fps）
合成结果：共 533 帧（约 17.8s）

注：帧差和SSIM反映的是智能切点选出的最优帧对的场景差异。三段视频拍摄场景不同，
场景差异较大；智能切点在首尾60帧中自动选取视觉最近的帧对，相比直接取首尾帧，
帧差从约41降至约32，切换更自然。

## 过渡质量说明：
- 帧间差：切点两帧的像素平均绝对差，越小说明场景越相近、切换越自然
- SSIM：结构相似性，越高说明衔接处视觉越连贯
- 过渡帧数（5帧≈167ms），在平滑过渡与避免幻灯片感之间取得平衡

## 使用方法：
1. 将北航校园视频片段（.mp4）放入 videos/ 目录
2. 运行 python main.py
3. 合成结果保存至 output/synthesized.mp4
